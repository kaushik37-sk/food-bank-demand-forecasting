# AI-Based Food Bank Demand Forecasting System

## Overview
This project aims to develop a data-driven system that helps food banks forecast demand and improve inventory and distribution planning. The goal is to support operational decision-making by identifying demand trends and potential shortage periods using historical data.

## Problem Statement
Food banks often rely on historical averages or manual planning when preparing for demand fluctuations. This can lead to shortages during peak periods or excess waste during low-demand periods. There is a need for a simple, interpretable forecasting tool that supports staff planning without increasing workload.
